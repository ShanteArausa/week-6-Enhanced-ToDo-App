## Overview
This is a mini-project demonstrating JavaScript mental models:
- **Scope**: block vs function vs global variables
- **Execution flow**: event listeners and DOM updates
- **Debugging**: console logs and DevTools

## Features
- Add tasks dynamically
- Mark tasks as complete/incomplete
- Filter tasks (All, Active, Completed)
- Persistent storage using localStorage

## Tech Used
- HTML5 (semantic)
- CSS3 (styling, hover effects)
- JavaScript (DOM, event listeners, app state)

## How to Run
1. Clone this repository
2. Open `index.html` in a browser
3. Interact with the To-Do List
